# Azure DevOps Cloud API FullModel (Celion Edition)

Dieses Paket beschreibt die vollständige REST API von Azure DevOps Services (Cloud).

- **Produkt:** Azure DevOps Services (Cloud)
- **Version:** v7.1-preview.7
- **Base URL:** https://dev.azure.com/{organization}/_apis/
- **Auth:** Personal Access Token (PAT) via Basic Auth
- **Stand:** Oktober 2025

## Enthaltene Module
- Projects & Teams
- Work Items
- Boards & Backlogs
- Repositories
- Pipelines
- Builds
- Releases
- Artifacts
- Users & Groups
- Test Management
- Service Hooks
- Analytics
- Extensions
