# Targetprocess On-Premise API FullModel (Celion Edition)

Dieses Paket beschreibt die REST API von **Targetprocess Server 2022.2 (On-Premise)**,
kompatibel zur Cloud-Version.

- **Produkt:** Targetprocess Server (On-Premise)
- **Version:** 2022.2 (LTS)
- **API-Version:** v1.1
- **Base URL:** https://{server}/api/v1/
- **Auth:** Bearer Token oder Basic (username:password)
- **Stand:** Oktober 2025

## Enthaltene Module
- System & Organisation
- Work Items
- Planning & Iterations
- Boards & Processes
- Relations & Dependencies
- Test Management
- Comments & Attachments
- Time Tracking
- Users & Roles
- Administration
- Reports & Views
- Integrations & Automations

> Hinweis: Die API-Struktur ist identisch zur Cloud-Version.
> Für On-Prem Installationen können Authentifizierung und Hosting-Setup abweichen.
